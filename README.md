# shop-review
네이버 쇼핑 리뷰 데이터를 활용한 한국어 감성 분석 프로젝트 | Korean Sentiment Analysis using Naver Shopping Reviews  
🛍️ 네이버 쇼핑 리뷰 데이터셋을 활용하여 긍정/부정 감성을 분류하는 자연어 처리 모델 
- Soylnp를 활용한 한국어 텍스트 전처리
- 머신러닝 기반 감성 분석 모델 구현
- 20만 건 이상의 리뷰 데이터 학습
